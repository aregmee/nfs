# Need For Speed - A car racing game (using JS and CSS)

Need for Speed using Javascript(JS) and CSS only.
Play here: https://ashimregme.github.io/need-for-speed

## Controls

```
Space -> Fires weapon
Left and Right Arrows -> Avoid Obstacle
```
